Руководство по использованию приложения контакты:
Вы можете:
    добавить контакт
    удалить контакт
    просмотреть добавленные контакты
    редактировать контакт
    осуществить поиск по добавленным контактам (вроде работает только строгий поиск(имя должно четко совпадать с имеющимся, по части не работает))

Требования:
    Postgresql с созданной за ранее бд (без заранее созданного не тестил, забыл)
    У вас должна быть установлена ОС Windows 10

Примечание:
    Данная программа была проверена на postgresql 16, параметры для входа были заданы:
        имя бд - phonebook, имя пользователя(он же админ(был использован по умолчанию)) - postgre,
        Пароль - 1234, порт на котором все должно быть запущено 1234.
